USAGE PATTERNS
================================================================================

FOR SINGLE STAGE PROGRAMME:
  1. Fill in Name of Provider
  2. Fill in Programme Title
  3. Fill in Stage (e.g., "1")
  4. Fill in Exit Award if applicable
  5. Mark delivery mode checkbox
  6. Mark teaching modalities checkboxes
  7. Mark assessment techniques checkboxes
  8. Add modules in rows 14-15+

FOR MULTI-STAGE PROGRAMME:
  1. Copy the entire table after row 11
  2. Update Stage field for each copy
  3. Mark appropriate checkboxes for each stage
  4. Fill in stage-specific modules

FOR DIFFERENT DELIVERY VERSIONS:
  Create separate tables for:
    • Full-time versions
    • Part-time versions
    • Blended versions
    • Online versions
    • Etc.

TROUBLESHOOTING
================================================================================

Q: Table doesn't paste correctly into Word
A: Ensure you copy ALL of the HTML (Ctrl+A), then paste as normal

Q: Rows don't align properly
A: Word may auto-adjust column widths. Use Table → AutoFit → AutoFit Window

Q: Merged cells appear broken
A: The HTML uses proper colspan/rowspan attributes. Word should render correctly.
   If not, you may need to manually recreate the merges using Table → Merge Cells

Q: Special checkbox symbol (✔) doesn't show
A: This is a Unicode character. Ensure your Word font supports it (Calibri does).
   If needed, substitute with [✓] or [X]

Q: How to add more rows for modules?
A: Copy rows 14-15 entirely and paste after row 15. Repeat as needed.
